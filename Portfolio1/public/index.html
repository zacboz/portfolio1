Grovemade Clone

Clone of eCommerce site www.grovemade.com that sells handcrafted wood accessories for apple products.

Team: Eric Simmons, Corey Harville, Gregory Luke, and Zac Boswell

Group project: We built a fully functioning clone of www.grovemade.com using Angularjs, Nodejs, Express, Gulp, Sass, ES6, ElephantSQL, and Stripe

Major challenge: The biggest challenge with this project was that each product was chalk full of its own individual detail, which involved many foreign keys. The logic of building our SQL tables to efficently return the information we needed proved difficult at many steps in the project.

Personal ownership: Building the backend and the logic of the our SQL tables, styling and building the functionality in the navbar, footer, shop view, cart view, and the checkout process with stripe functionality

  • Built: February 2017
  • See the project live @ grovemadeclone.com

Sustenance

A web application that utilizes the Yelp API to search for restaurants by location and where users can build custom restaurant collections.

Create your own collection of favorite restaurants!

Personal Project: Built a full stack application using Angularjs, Bootstrap, Nodejs, Express, Gulp, Sass, and PostgresSQL while a student at DevMountain. This web application utilizes the Yelp API to allow users to search for restaurants based off location, login and create custom collections of their favorite restaurants. The site is mobile responsive as well.

Major Challenge: This was my first time buidling a fully functional application with a backend and successfully applied create, read, update, and delete endpoints.

Github Repo

  • Built: January 2017
  • See the project live @ sustenance-app.herokuapp.com
